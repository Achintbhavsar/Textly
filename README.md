# 🚀 Textly – Real-Time Cross-Platform Chat Application

Textly is a cross-platform real-time messaging application developed for Android and Web. It enables seamless communication through instant messaging, group chats, voice calls, video calls, and push notifications, delivering a smooth and responsive user experience.

---

## ✨ Features

* 💬 Real-time messaging
* 👥 One-to-one and group chat
* 📞 Voice calling (WebRTC)
* 🎥 Video calling (WebRTC)
* 🔔 Push notifications
* 🔄 Real-time data synchronization
* ⚡ Fast and responsive UI
* 🔐 Secure authentication

---

## 🛠️ Tech Stack

| Category            | Technologies / Tools               |
| ------------------- | ---------------------------------- |
| Android Development | Kotlin, Jetpack Compose, WebRTC    |
| Frontend (Web)      | React.js, TypeScript, Tailwind CSS |
| Backend             | Node.js, Express.js, Socket.io     |
